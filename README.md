Aplicación de tareas que permite crear, completar, eliminar y filtrar tareas.

La aplicación tiene las siguientes secciones: Título, cabecera, filtros, lista de tareas, formulario para añadir una tarea y panel de estadísticas
El usuario podrá añadir tareas, modificar tareas, eliminar las tareas, marcas las tareas como completadas, filtrar las tareas que se muestran y ver estadísticas
